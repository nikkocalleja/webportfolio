Capstone Project 1
Frontend Design

A front-end page refers to the user-facing part of a website or web application that is directly visible and accessible to users in their web browsers. It encompasses the design, layout, content, and interactive elements that users interact with directly. Front-end pages are built using a combination of HTML (HyperText Markup Language) for structure, CSS (Cascading Style Sheets) for styling, and often JavaScript for interactivity, animations, and dynamic content. These pages are responsible for presenting information and providing a user-friendly interface for visitors to interact with the website or application.

Capstone Project 2
Backend Fundamentals

A back-end page refers to the behind-the-scenes, server-side part of a website or application that handles data processing, storage, and business logic. It manages databases, user authentication, and other server-related functions, supporting the functionality of the front-end interface that users interact with.

Capstone Project 3
Fullstack Development

A full-stack page is a complete web page that integrates both the front-end and back-end components, encompassing the user interface, functionality, data processing, and server interactions. It combines HTML, CSS, JavaScript for the front-end, and server-side technologies (such as databases and programming languages like Python, Ruby, or Node.js) for the back-end, providing a comprehensive solution for a web application or website.
